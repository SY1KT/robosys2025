# robosys2025 
ロボットシステム課題1 サイコロの目の出力  
標準入力からサイコロの個数と振る回数を入力してサイコロの目とその合計を出力する.

## 実行方法
```bash  
$ git clone <私のリポジトリURL> 
$ cd robosys2025 
$ ./kadai    

```
## 出力例
```bash
サイコロの個数nと振る回数mを入力:4 5
1回目: [1, 1, 1, 5] 合計: 8
2回目: [1, 6, 4, 2] 合計: 13
3回目: [1, 1, 6, 1] 合計: 9
4回目: [2, 5, 2, 5] 合計: 14
5回目: [5, 5, 4, 5] 合計: 19
```
注意:個数と回数を入力する所はスペースを空ける
