# robosys2025 
ロボットシステム課題1 サイコロの目の出力  
標準入力からサイコロの個数と振る回数を入力してサイコロの目とその合計を出力する.

## 実行方法
```bash  
$ git clone <git@github.com:SY1KT/robosys2025.git> 
$ cd robosys2025 
$ ./dice    
```
